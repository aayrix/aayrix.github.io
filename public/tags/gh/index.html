GitHub CLI (gh) — Complete Guide with Copy-Paste Commands

The GitHub CLI (gh) lets you interact with GitHub directly from your terminal — create repos, open PRs, review issues, and more without leaving the command line. This guide covers installation, authentication, and the most useful commands. Prerequisites: A GitHub account and Git installed on your system.
